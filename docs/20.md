## 问题

1.  真实世界的对象包含 ___ 和 ___。
2.  软件对象的状态存储在 ___ 中。
3.  软件对象的行为通过 ___ 暴露。
4.  隐藏来自外部世界的内部数据，并且仅通过公开暴露的方法访问它被称为 data ___。
5.  软件对象的蓝图称为 ___。
6.  常见行为可以在 ___ 中定义，并使用 ___ 关键字继承到 ___。
7.  没有实现的方法集合称为 ___。
8.  按功能组织类和接口的命名空间称为 ___。
9.  API 这个词代表 ___？
