# 课程：安全功能概述

> 原文： [https://docs.oracle.com/javase/tutorial/security/overview/index.html](https://docs.oracle.com/javase/tutorial/security/overview/index.html)

安全功能可帮助您保护程序和数据免受损害，保护数据屏蔽和私有，以及在安全感知运行时环境中部署新应用程序。

Java 还提供了一些工具来帮助您管理对系统资源的访问;创建，存储和维护加密的公钥和私钥（密钥对）和证书;并在部署过程中创建和签署 jar 文件。

* * *

**Note:** For a brief overview of security features supported in Java SE 7, see the [Security Overview](https://docs.oracle.com/javase/8/docs/technotes/guides/security/overview/jsoverview.html) Whitepaper in the Security Guides directory.
